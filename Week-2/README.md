# Calculator API

A simple calculator API calculates basic arithmetic operations.

## Installation

1. Clone this repository
2. Run `npm install`
3. Start the server with `npm start`

## Available Endpoints

- GET `/add?a=<number>&b=<number>` - Adds two numbers
- POST `/multiply` - Multiplies two numbers
